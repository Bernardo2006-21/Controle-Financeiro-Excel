# 📊 Controle Financeiro Mensal em Excel

> Dashboard em Excel para análise de receitas e despesas

## 📌 Projeto
Dashboard financeiro desenvolvido no Excel para controle de receitas e despesas.

## 🚀 Funcionalidades
- Controle de receitas e despesas  
- Tabela dinâmica  
- Gráfico automático por categoria  
- Cálculo de saldo mensal  

## 🛠️ Ferramentas
- Microsoft Excel  
- Tabelas dinâmicas  
- Dashboard  

## 📎 Arquivo
Baixe o arquivo Excel neste repositório para utilizar.
